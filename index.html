Java Entry Point

The main() method is the entry point for a Java application. run() is typically used for new threads or tasks.

it is categorized by its (String[]arg) as a parameter to the function 

    
      public class MyClass{
      public static void main(String args[])
      }

Here’s an explanation of the different components of the entry point method:

  1. public: It specifies that the entry point method is accessible from outside the class.
  2. static: It indicates that the entry point method belongs to the class itself rather than an instance of the class.
  3. void: It specifies that the entry point method does not return any value.
  4. main: It is the name of the method. This name is recognized by the JVM as the entry point.
  5. String[] args: It is the parameter passed to the entry point method. It represents the command-line arguments passed to the program.

Object Oriented Programing Langauge

Java is an object-oriented programming language.

When the individual objects are created, they inherit all the variables and methods from the class.

Procedural programming is about writing procedures or methods that perform operations on the data, while object-oriented programming is about creating objects that contain both data and methods.

          
            public class Bird {
              …
              Public void sound ( ) {
              System.out.println ( “ birds sounds “ );
              }
              }
              public class pigeon extends Bird {
              …
              @override
              public void sound ( ) {
              System.out.println( “ cooing ” ) ;
              }
              }
              public class sparrow extends Bird ( ) {
              ….
              @override
              Public void sound ( ){
              System.out.println( “ chip ” ) ;
              }
              }
